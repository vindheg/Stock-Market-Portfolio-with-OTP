#  Stock Market Portfolio (Spring Boot + HTML/Tailwind)

A simple **stock portfolio simulation web app** built using:

- **Spring Boot (Backend)**
- **HTML, CSS, Tailwind (Frontend)**

Users can:

Login
Buy and sell stocks  
See portfolio balance and holdings updated live  
View market stock prices (mock data)  


## Features

| Feature | Description |
|---------|-------------|
| Login screen | Username stored in sessionStorage |
| Buy / Sell stocks | Updates balance + holdings dynamically |
| Live frontend updates | Fetch API calls backend, updated data returned |
| In-memory storage | Portfolio maintained in backend while app runs |
| Mock market prices | NSE / BSE sample stock prices |
| Ready for DB upgrade | Design is service + controller layer structured |
